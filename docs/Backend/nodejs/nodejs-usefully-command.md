## 怎样升级node版本

### [Linux/Mac](https://stackoverflow.com/questions/10075990/upgrading-node-js-to-latest-version)

```bash
sudo npm install n -g
sudo n 0.12.2
```

升级到稳定版本:
```bash
sudo n stable
```
升级到最近版本：
```bash
sudo n latest
```

### Windows

从node官网下载.msi重新安装即可
