## intellij 嵌入打开bash和弹窗打开bash方式
嵌入打开bash配置路径为
"C:\Program Files\Git\bin\sh.exe"  -login -i
弹窗方式打开路径为
"C:\Program Files\Git\git-bash.exe"
之所以用弹窗方式，是因为bash的快捷键与intellij有冲突
